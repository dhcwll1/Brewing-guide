Modifiers

Using certain ingredients can modify the potion

Using fermented spider eye will revert/corrupt the potion.

Using gunpowder will make it a throwable potion.

Using dragon's breath on throwable potion will make it a lingering potion.

Using redstone powder will make the effect last longer, and using glowstone powder will make it stronger. You can't use them at the same time.
